## 🚀 Quick Start

### 1. Access the CMS

```
http://localhost:3000/cms/portfolio
```

### 2. Choose a Section

Click any of the 8 tabs to manage that section

### 3. Manage Data

- **Add**: Click "Tambah" button → Fill form → "Simpan"
- **Edit**: Click "Edit" on any item → Update → "Simpan"
- **Delete**: Click "Hapus" on any item → Confirm
- **Search**: Use search box to filter
