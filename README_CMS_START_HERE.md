### 2. Pilih Section (dari 8 tab)

- **Info** - Update profil pribadi

- **Kerja** - Pengalaman kerja
- **Pendidikan** - Riwayat pendidikan
- **Keahlian** - Technical skills
- **Sertifikat** - Sertifikasi
- **Bahasa** - Bahasa yang dikuasai
- **Testimoni** - Review klien

### 3. Kelola Data

- **Tambah** → Klik tombol "Tambah"
- **Edit** → Klik tombol "Edit"
- **Hapus** → Klik tombol "Hapus"
- **Cari** → Gunakan search box
